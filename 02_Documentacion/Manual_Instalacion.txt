MANUAL DE INSTALACIÓN - PROYECTO MI TIENDA PRO
1. Requisitos Previos
Tener instalado Node.js (versión 16 o superior).

Tener instalado pnpm (o npm/yarn).

Cuenta de Stripe (para obtener las claves de prueba).

Base de datos en MongoDB Atlas o local.
********************************************************
2. Configuración del Backend
Abrir una terminal en la carpeta /backend.

Instalar dependencias: npm install.

IMPORTANTE: Crear un archivo llamado .env en la raíz de la carpeta /backend.

Dentro del archivo .env, configurar las siguientes variables:

Fragmento de código

PORT=5000
MONGO_URI=tu_cadena_de_conexion_a_mongodb
STRIPE_SECRET_KEY=tu_clave_secreta_de_stripe_sk_test_...
Iniciar el servidor: node server.js o npm start.
********************************************************
3. Configuración del Frontend
Abrir una terminal en la carpeta /frontend.

Instalar dependencias: pnpm install (o npm install).

Iniciar la aplicación: pnpm run dev (o npm run dev).

Abrir en el navegador: http://localhost:5173.
*********************************************************
4. Notas de Seguridad
El archivo .env ha sido excluido del repositorio de Git para proteger las claves de Stripe.

Nunca compartir la STRIPE_SECRET_KEY en entornos públicos.

Asegurarse de que el puerto 5000 no esté siendo usado por otra aplicación antes de iniciar el backend.
